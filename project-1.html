WeColor App

Crea, likea y explora las mejores paletas de colores. Permite copiar colores y guardar las paletas. También exportarlas como pdf.

Link deploy

Project Overview

En esta App Web el usuario puede crear su cuenta e iniciar sesión para comenzar a crear las paletas. El usuario puede crear las paletas, elegir los colores, copiar su código HEX, Explorar paletas creadas por otros usuarios, Al momento de crear la paleta el usuario añade un titulo, descripción y tags. El usuario puede likear paletas de colores y luego filtrar en el explorador por las paletas con mayor cantidad de likes. Esta aplicación se desarrolló con React Js y, NodeJs y MongoDB para el Backend. Utilicé la librería Material UI para la construcción de componentes de interfaz de usuario.

El manejo de estado dentro de la App se realiza utilizand Redux.
