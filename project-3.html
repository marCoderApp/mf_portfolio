Project Overview

Esta API REST fue construida para servir a el cliente de WeColor. Se utilizó Javascript y NodeJs, con un patrón MVC (Modelo, Vista, Controlador) y MongoDB para la persistencia de datos en una base de datos no relacional. Actualmente se encuentra desplegada en Render.com y es la url que utiliza la aplicación del cliente (React.js) para realizar las peticiones y mostrar al usuario la información requerida.
