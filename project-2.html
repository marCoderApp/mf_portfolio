GIST (Gestión Informática de Servicios Técnicos)

Aplicación desktop desarrollada en Java para la gestion informatica de talleres de servicios técnicos.

Descarga

Project Overview

GIST (Gestión Informática de Servicios Técnicos) es una App Desktop. Programada enteramente en Java. Se usa JavaFX para la realización de la interfaz gráfica y MySQL JDBC para la persistencia de datos. Con este software el Administrador puede registrar ordenes de trabajo, listarlas, ver sus detalles y generar consultas y asociarlas por id de orden de trabajo. El usuario puede asignar presupuestos y detalles de reparación a cada orden de trabajo. La aplicacion ya se encuentra operativa en su primera versión. Utiliza un servidor de bases de datos MySQL desplegado en Railway.com. En el futuro se podrá acceder a la aplicación desde dispositivos moviles.

Se creó un instalador .msi para dispositivos Windows. Tan solo es necesario instalar el programa y colocar las credenciales correctas para que el software esté listo para su uso.

Este proyecto comenzó como parte de una asignatura en la universidad. Tambien se realizó el proceso de análisis de sistema incluyendo elicitación, diagrama de casos de uso, diagramas de secuencia, diagramas de dominio, casos de prueba, etc.
